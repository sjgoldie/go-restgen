# go-restgen v0.1.0

Initial release of go-restgen - a lightweight, type-safe REST API framework for Go using generics.

## Features

- 🚀 **Zero boilerplate CRUD** - Generate full REST APIs with a single line
- 🔒 **Type-safe** - Leverage Go generics for compile-time type checking
- 🎯 **Nested resources** - First-class support for parent-child relationships
- 🏗️ **Production-ready** - Built on Chi router and Bun ORM
- 📦 **Composable** - Mix generated routes with custom handlers
- 🧪 **Well-tested** - 97.8% coverage on router, comprehensive test suite

## Quick Start

```go
import "github.com/sjgoldie/go-restgen/router"

// Define your model
type User struct {
    bun.BaseModel `bun:"table:users"`
    ID            int    `bun:"id,pk,autoincrement" json:"id"`
    Name          string `bun:"name,notnull" json:"name"`
}

// Register CRUD routes
router.RegisterCRUD[User](r, dbServer, "/users")

// You now have full CRUD: GET, POST, PUT, DELETE
```
